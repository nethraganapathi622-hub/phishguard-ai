# 🛡️ PhishGuard AI – Fraud & Phishing Detection System

## 📌 Overview

PhishGuard AI is an industry-oriented project that detects fraudulent transactions and phishing activities using Machine Learning. It analyzes transaction patterns and predicts whether a transaction is **fraudulent or normal**.

---

## 🚀 Features

* 🔍 Fraud detection using ML models
* 📊 Data preprocessing and feature engineering
* 🤖 Model training and prediction
* 🌐 Interactive UI (Streamlit / React frontend)
* ⚡ Real-time prediction system

---

## 🛠️ Tech Stack

* Python
* Pandas, NumPy
* Scikit-learn
* Streamlit
* React (Frontend)
* Tailwind CSS

## ⚙️ Installation & Setup

### 1️⃣ Clone the repo

```
git clone https://github.com/nethraganapathi622-hub/phishguard-ai.git
cd phishguard-ai
```

### 2️⃣ Backend (ML)

```
cd ml
pip install -r requirements.txt
streamlit run app.py
```

### 3️⃣ Frontend

```
npm install
npm run dev
```
